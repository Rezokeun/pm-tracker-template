# validation/

Captured validations — one file per item (the confirming artifact: a signed doc, an approval, a screenshot, a passing test). Name each `YYYY-MM-DD-<slug>` (e.g. `2026-06-25-vendor-msa.pdf`). Referenced from the `validation` column in `tracker.md` and from `[done]` lines in `session-log.md`.

This is the "trust but verify" record: when someone asks for confirmation later, it's already here — no hunting.
