# Source Log

Append-only, **newest on top**. The **verbatim** record of raw input the owner provides — feedback, updates, data, context — captured *before* it's summarized into notes or tracker items. This is provenance: read it by date or search when you need to verify or re-derive something a summary may have lost. It is **not** a browsable layer (that's `notes.md`) and **not** an event digest (that's `session-log.md`).

**Capture rule** (see `CLAUDE.md` → Logging): save **all** raw input verbatim and datestamped, *before* summarizing. Never judge importance — when in doubt, capture. Each curated note or tracker item references the dated entry it came from.
